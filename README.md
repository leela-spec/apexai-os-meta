# ApexAI_OS Meta Repo

This repository contains the ApexAI_OS Meta Operating Core.

It is the source for:

- managed operating law
- project interface templates
- promotion and QA/Hygiene protocols
- reusable holding-layer agent seeds
- versioned meta releases
- sanitized learning intake

## Boundary

This repo does not contain raw project data, private operator memory, credentials, or server-local OpenClaw runtime config.

Hetzner/OpenClaw clone binding is deferred until after the meta repo and first project repo exist.
