# Define safety and scope boundaries for NARM-support system

## Acceptance Criteria

- Boundaries state that the system supports therapy and does not replace therapist judgment.
- Outputs include disclaimers against diagnosis or treatment claims.
- Sensitive personal material handling rules are explicit.

## Definition of Done

- Safety and scope policy is ready for operator review.
- Therapist-facing and self-facing output limits are documented.

## Notes

- This task anchors all downstream design choices.
