# Design compact NARM therapist session-prep output format

## Acceptance Criteria

- Output format is compact enough to bring to a NARM therapist.
- Format separates observations, source references, questions, themes, and desired therapist support.
- Format avoids presenting generated interpretations as facts.

## Definition of Done

- Session-prep template is ready for review.

## Notes

- Therapist-prep outputs may need redaction rules before use.
