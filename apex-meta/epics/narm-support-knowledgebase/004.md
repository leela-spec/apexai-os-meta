# Design personal psychological material index structure

## Acceptance Criteria

- Personal files have proposed index categories.
- Index distinguishes raw source, interpretation, recurring pattern, therapist-relevant note, and uncertainty.
- Relationship-pattern material is included if source evidence exists.

## Definition of Done

- Personal-material index design is ready for review.

## Notes

- Personal material must not be over-interpreted as clinical fact.
