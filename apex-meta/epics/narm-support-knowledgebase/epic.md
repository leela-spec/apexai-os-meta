# NARM-Support Therapy Knowledgebase Infrastructure

## Goal

Build a NARM-support infrastructure that indexes NARM theory and personal psychological material, creates guided self-exploration flows aligned with NARM, and produces compact session-prep outputs for a NARM therapist.

## Scope

- Index NARM theory from `ET-Heller-NARM.md`.
- Index personal psychological material from anamnesis, shadow insights, relationship-pattern notes, and therapy handover files.
- Create guided self-exploration flows aligned with NARM.
- Produce compact session-prep outputs for a NARM therapist.

## Constraints

- This is a therapy-support knowledgebase, not a replacement for a therapist.
- Outputs must not claim diagnosis, treatment, or clinical authority.
- Raw source material, interpretation, uncertainty, and therapist-facing summary must remain distinguishable.
