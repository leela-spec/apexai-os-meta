# Smoke Product

The Smoke Product is a synthetic fixture used only to verify the Apex KB lifecycle.

## Ownership

It owns a catalog of named widgets and their active or retired status. It does not own purchasing, user accounts, billing, or deployment.

## Relationships

A Catalog contains Widgets. Each Widget has exactly one Status. Status is either Active or Retired.

## Current state

Widget Alpha is Active. Widget Beta is Retired. No contradictions or unresolved product questions are known in this fixture.
