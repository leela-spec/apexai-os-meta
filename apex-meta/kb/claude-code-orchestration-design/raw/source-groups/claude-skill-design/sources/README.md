# Sources

KB-specific working source corpus for `claude-skill-design`.

## Folder roles

- `curated/` — sources copied or promoted from raw acquisition archives after relevance review.
- `operator-supplied/` — additional files manually supplied by the operator.

## Placement rule

If unsure where a new source belongs, place it in:

```text
operator-supplied/uncategorized/
```

The Phase 1 index/classification pass should classify it before synthesis.
