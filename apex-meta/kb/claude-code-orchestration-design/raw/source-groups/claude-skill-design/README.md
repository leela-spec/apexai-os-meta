# Claude Skill Design

Dedicated Apex KB workspace for Claude / Agent Skills / SKILL.md / skill-package design research.

## Purpose

This KB is for building an evidence-grounded knowledge base about Claude skill design, skill package architecture, progressive disclosure, routing descriptions, support-file boundaries, script/tool boundaries, examples, validation, and security/trust concerns.

## Boundary

- This is the target KB / knowledge product.
- `_source-acquisitions/` remains the raw intake/archive area.
- `.claude/skills/apex-kb/` is the runner/process package, not the evidence corpus for this KB.
- `apex-kb-skill-test/` is not part of this KB.

## Main folders

- `sources/` — KB-specific working source corpus.
- `ingest-analysis/` — Phase 1 source-level analysis outputs.
- `wiki/` — Phase 2 compiled KB pages after operator approval.
- `manifests/` — KB-specific source inventories, allowlists, and run ledgers.
- `reports/` — audits, validation reports, and correction reports.
