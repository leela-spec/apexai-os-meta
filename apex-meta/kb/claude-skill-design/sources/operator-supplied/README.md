# Operator-Supplied Sources

Drop additional Claude skill design sources here.

## Subfolders

- `official/` — official docs or vendor materials supplied manually.
- `examples/` — SKILL.md examples, packages, snippets, and implementation evidence.
- `academic/` — papers, abstracts, research notes.
- `secondary/` — indexes, summaries, explainers, maps.
- `notes/` — operator notes and chat-derived planning material.
- `uncategorized/` — default drop zone when classification is unclear.

## Default rule

If unsure, place files in `uncategorized/` and let Phase 1 classify them.
