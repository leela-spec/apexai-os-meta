Title: So rufen Sie alle Finanzinformationen zu einem Event in einer Event-Rechnung ab | Eventbrite-Hilfecenter

URL Source: https://www.eventbrite.de/help/de/articles/263015/so-koennen-sie-alle-finanzinformationen-zu-einem-event-in-form-einer-event-rechnung-abrufen/

Markdown Content:
Laden Sie eine Rechnung mit Event- oder Auszahlungsdetails herunter, um einen Überblick über die Gebühren, Steuern und Einnahmen zu erhalten, die für Ihr Event generiert wurden. Navigieren Sie im Event-Dashboard zu „Bestellungen“ (unter „Teilnehmer verwalten“), um mit der Bearbeitung zu beginnen. Klicken Sie dann auf „Event-Rechnung“ oder „Auszahlungsdetails“.

In diesem Artikel

*   1. Gehen Sie zu „Meine Events verwalten“.

*   2. Rufen Sie Ihr Event-Dashboard auf.

*   3. Klicken Sie auf Bestellungen (unter „Teilnehmer verwalten“).

*   4. Optional: Klicken Sie auf „Event-Rechnung“.

*   5. Optional: Klicken Sie auf „Auszahlungsdetails“.

**HINWEIS:** Event-Rechnungen sind nur für Events verfügbar, die in den Währungen EUR, GBP, CAD, AUD oder NZD verkauft werden. Wenn Ihr Event eine andere Währung verwendet, gehen Sie stattdessen zum eblink-Bereich [Verkaufsübersicht](https://www.eventbrite.de/help/de/articles/192276/).

## 1. Gehen Sie zu „Meine Events verwalten“.

Melden Sie sich bei Ihrem Eventbrite-Konto an und wählen Sie [„Meine Events verwalten“](https://www.eventbrite.de/organizations/events) aus dem Kontomenü aus.

## 2. Rufen Sie Ihr Event-Dashboard auf.

Klicken Sie auf Ihr Event, um das Event-Dashboard aufzurufen.

## 3. Klicken Sie auf Bestellungen (unter „Teilnehmer verwalten“).

## 4. Optional: Klicken Sie auf „Event-Rechnung“.

Die Event-Rechnung enthält eine Übersicht über die für Ihr Event erhobenen Gebühren und Steuern und steht als PDF zur Verfügung. Der Name auf Ihrer Rechnung wird aus Ihrem Veranstalterprofil und die Adresse aus Ihrer Auszahlungsmethode abgerufen. Event-Rechnungen können nicht angepasst werden.

## 5. Optional: Klicken Sie auf „Auszahlungsdetails“.

Die Rechnung mit den Auszahlungsdetails ist als PDF verfügbar. Sie umfasst:

*   Gesamteinnahmen

*   Eventbrite-Gebühren

*   Erhobene Steuer

*   Nettoeinnahmen

Der Name auf Ihrer Rechnung wird aus Ihrem Veranstalterprofil und die Adresse aus Ihrer Auszahlungsmethode abgerufen. Rechnungen mit Auszahlungsdetails können nicht angepasst werden.

**TIPP:**Sie müssen Adobe Reader herunterladen, um PDF-Dateien öffnen und anzeigen zu können.
