Markdown Content:
Im Bericht "**Umsatzübersicht**" erhalten Sie einen Überblick über Ticketumsatz, Ticketbuchungsgebühren und Steuern Ihres Events. Wählen Sie Filter und Spalten aus, um Ihrem Bericht die entsprechenden Daten hinzuzufügen. Anschließend können Sie die Daten nach den Aspekten gruppieren, die Ihnen am wichtigsten sind. Gehen Sie zunächst zum Event-Dashboard und wählen Sie "**Berichterstellung**". Wählen Sie dann unter "**Eventberichte**" die Option "**Umsatzübersicht**". Um diesen Bericht für mehrere Events herunterzuladen, gehen Sie unter "Organisation bearbeiten" zu "**Berichterstellung**".

In diesem Artikel

*   Laden Sie eine Verkaufsübersicht für eine Veranstaltung herunter

*   Laden Sie eine Verkaufsübersicht für mehrere Veranstaltungen herunter

**HINWEIS:** Für den Zugriff auf den Bericht "Umsatzübersicht" benötigen Sie die Berechtigung zum Abrufen "erweiterter Berichte". Wenn Sie nicht selbst der Kontoinhaber sind, bringen Sie Ihre Berechtigungen bitte beim Verantwortlichen in Erfahrung.

## Laden Sie eine Verkaufsübersicht für eine Veranstaltung herunter

### 1. Rufen Sie Ihr Event-Dashboard auf.

[Melden Sie sich bei Ihrem Eventbrite-Konto an](https://www.eventbrite.de/signin) und gehen Sie zu "[Events](https://www.eventbrite.de/organizations/events)". Wählen Sie Ihr Event aus.

### 2. Gehen Sie zu "Berichterstellung" und wählen Sie "Eventberichte".

### 3. Wählen Sie "Umsatzübersicht" (unter "Verkäufe").

### 4. Optional: Wählen Sie einen Datumsbereich.

Wählen Sie einen Zeitraum aus, für den Sie den Umsatz abrufen möchten.

*   Seit Verkaufsbeginn

*   Diese Woche

*   Diesen Monat

*   Dieses Jahr

*   Benutzerdefinierter Zeitraum

### 5. Optional: Fügen Sie zusätzliche Daten hinzu.

Unter "Umsatzübersicht" können Sie verschiedene Kästchen ankreuzen:

*   Zusatzartikel in den Bericht aufnehmen: Gesamtanzahl der gekauften Zusatzartikel

*   Gebühren und Nutzungsgebühren in den Bericht aufnehmen: Gesamtbetrag der erhobenen Gebühren und/oder Nutzungsgebühren

*   Bruttoticketumsatz anzeigen: Gesamtumsatz inklusive Ticketpreis plus aller Gebühren

*   Kostenlose Tickets anzeigen: Gesamtzahl der kostenlosen Tickets

### 6. Exportieren Sie Ihre Daten.

Wählen Sie einen Dateityp aus und geben Sie an, wie der Bericht exportiert werden soll. Folgende Exportformate stehen zur Auswahl:

*   CSV- bzw. .csv-Datei

*   Excel- bzw. .xls-Datei

*   Druckvorschau oder .pdf-Datei

Sobald Sie den Dateityp ausgewählt haben, wird der Bericht lokal heruntergeladen.

## Laden Sie eine Verkaufsübersicht für mehrere Veranstaltungen herunter

### 1. Gehen Sie zu "Berichterstellung".

[Melden Sie sich bei Ihrem Eventbrite-Konto an.](https://www.eventbrite.de/signin) Wählen Sie anschließend "**Berichterstellung**" aus.

### 2. Wählen Sie "Umsatzübersicht" (unter "Verkäufe").

### 3. Geben Sie an, welche Events in den Bericht aufgenommen werden sollen.

Wählen Sie die Events aus und dann "**Bericht ausführen**". Sie können nach allen, bevorstehenden oder vergangenen Events filtern oder anhand des Namens nach Events suchen.

### 4. Optional: Wählen Sie einen Datumsbereich.

Wählen Sie einen Zeitraum aus, für den Sie den Umsatz abrufen möchten.

*   Seit Verkaufsbeginn

*   Diese Woche

*   Diesen Monat

*   Dieses Jahr

*   Benutzerdefinierter Zeitraum

### 5. Optional: Fügen Sie zusätzliche Daten hinzu.

Unter "Umsatzübersicht" können Sie verschiedene Kästchen ankreuzen:

*   Zusatzartikel in den Bericht aufnehmen: Gesamtanzahl der gekauften Zusatzartikel

*   Gebühren und Nutzungsgebühren in den Bericht aufnehmen: Gesamtbetrag der erhobenen Gebühren und/oder Nutzungsgebühren

*   Bruttoticketumsatz anzeigen: Gesamtumsatz inklusive Ticketpreis plus aller Gebühren

*   Kostenlose Tickets anzeigen: Gesamtzahl der kostenlosen Tickets

*   Versand anzeigen: Gesamtbetrag der für den Versand berechneten Gebühren

### 6. Exportieren Sie Ihre Daten.

Wählen Sie einen Dateityp aus und geben Sie an, wie der Bericht exportiert werden soll. Folgende Exportformate stehen zur Auswahl:

*   CSV- bzw. .csv-Datei (empfohlen für Berichte mit mehr als 1 Million Datensätze)

*   Excel- bzw. .xls-Datei

*   Druckvorschau oder .pdf-Datei

Sobald Sie den Dateityp ausgewählt haben, wird der Bericht automatisch heruntergeladen.
