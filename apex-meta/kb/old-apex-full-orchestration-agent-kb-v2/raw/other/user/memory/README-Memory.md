# Memory Area

This area stores the segmented memory system.

- `MEMORY.md` is the lightweight index.
- `MISTAKES.md` tracks durable mistake patterns and standing rules.
- `memory-daily/` stores dated daily logs.
- `memory-people/` stores person detail files.
- `memory-projects/` stores project detail files.
- `decisions/` stores monthly decision logs.
- `context/active.md` holds the current active context snapshot.

Keep the index lightweight and move durable detail into the matching subfolder.
