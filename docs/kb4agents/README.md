# KB4Agents Source Pack Archive

This directory preserves historical source and evidence artifacts from KB4Agents research and screening work.

These files are archival source-pack material. They are not managed KB canon, and they are not runtime truth. Accepted and maintained doctrine lives under `managed/`.

Local absolute paths and operator-local filesystem details have been sanitized before archival. Preserve filenames, source roles, rankings, read modes, validation reports, prompts, and patch artifacts unless they expose private filesystem paths, credentials, or other sensitive raw references.

Do not promote these files into `managed/` or add them to runtime indexes unless a future promotion pass explicitly reviews and accepts them.
